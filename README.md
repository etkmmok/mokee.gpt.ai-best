# mokee.gpt.ai-best
## 소개
이 프로젝트는 교사와 교육자를 위한 AI 에듀테크 도구 및 프롬프트 모음으로, 수업 준비부터 학습 자료 제작까지 다양한 교육 활동을 더욱 쉽고 효율적으로 수행할 수 있도록 돕습니다. AI를 활용하여 교실 수업의 생산성을 높이고 학생들의 학습 경험을 향상시키는 것을 목표로 합니다.
## 주요 기능 
- AI를 활용한 수업 자료 및 활동 생성
- 영어 수업을 위한 프롬프트 및 템플릿 제공
- 학습지, 퀴즈, 평가 문항 자동 생성 지원
- 교사의 업무 효율성을 높이는 다양한 AI 활용 도구 제공
## 사용방법

2. 실행
